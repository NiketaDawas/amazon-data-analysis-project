# Amazon Product Data Analysis Project

This project performs data analysis on Amazon product dataset using Python.

## Objectives
- Analyze pricing patterns
- Study discount trends
- Analyze customer ratings
- Visualize insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Amazon product dataset containing 1465 records and 16 columns.

## Key Insights
- Most products have ratings between 4 and 5
- Discounts significantly reduce prices
- Price does not strongly affect customer ratings
